# pixi-whitealbum

使用[PixiJS](https://github.com/pixijs/pixi.js)制作的[白色相簿2](http://touma.whitealbum.cc/wa/)游戏

![](./qrcode.jpg)

[国外访问](https://anata.me/pixi-whitealbum/wa2/)

### 安装
```bash
npm install
```

### 运行
```bash
npm run dev
```
访问`http://localhost:8081`, 使用手机模拟器查看效果

### 打包
```bash
npm run build
```
查看`dist`目录

### Github Pages
[git actions](./.github/workflows/ci.yml) 自动发布
